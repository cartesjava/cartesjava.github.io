    Procédure: installer le JDK 11 #

    • Si ce n’est pas déjà fait, créer un compte Oracle: https://profile.oracle.com/myprofile/account/create-account.jspx

    • Visiter https://www.oracle.com/java/technologies/javase/jdk11-archive-downloads.html

    • Télécharger la version Windows x64 Installer

      • jdk-11.0.16_windows-x64_bin.exe

    • Accepter la licence et je clique sur Download

    • Exécuter jdk-11.0.16_windows-x64_bin.exe

    • Installer avec les options par défaut
