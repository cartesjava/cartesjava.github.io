    Procédure: installer Eclipse #

    Rappel: ne pas effectuer cette étape sur un Windows du Collège


    1. Recomandé: désinstaller toute ancienne version d’Eclipse

    2. Installer Eclipse (version 2022-06 ou plus récent)

      • Visiter https://www.eclipse.org/downloads/
      • Télécharger en cliquant sur Download x86_64
      • Exécuter eclipse-inst-win64.exe
      • Installer Eclipse IDE for Java Developers avec les options par défaut

    Configurer Eclipse #

    1. Je démarre Eclipse

      • Je choisi mon WORKSPACE, p.ex.
        • Z:\eclipse-workspace (Windows du Collège)
        • C:\Users\PRENOM_NOM\eclipse-workspace (Mon Windows)

    2. Je m’assure que Eclipse utilise le JDK 11

      • Window => Preferences

      • Java => Installed JREs

      • Idéallement, Eclipse utilise uniquement le JDK 11

      • Si j’en ai pas besoin, je fait Remove sur tous les JDK sauf 11

      • Si le JDK 11 n’est pas là, je l’ajoute:

        • Add =>
        • Standard VM => Next
        • Sélectionner le répertoire racine du JDK, p.ex:
          • C:\Program Files\Java\jdk-11.XXX
        • Finish
        • Apply and Close

      • Je m’assure que le JDK 11 est le défaut (en gras)
